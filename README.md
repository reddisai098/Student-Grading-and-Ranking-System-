# Student-Grading-and-Ranking-System-
A terminal-based C++ application that manages student records, automatically calculates grades and average scores, tracks data persistence via file handling, and ranks students on a dynamic leaderboard using custom sorting algorithms.

A robust Command Line Interface (CLI) application built in C++ designed to manage, track, and rank student performance.

## 🚀 Features
* **Data Persistence:** Automatically saves and loads data to/from a local `students.txt` file.
* **Smart Ranking:** Sorts and displays a real-time leaderboard from highest to lowest average score.
* **Input Validation:** Gracefully handles unexpected user inputs without crashing.

## 🛠️ Core Concepts Practiced
* Object-Oriented Programming (Classes & Structs)
* File Stream Handling (`<fstream>`)
* Vector manipulation and dynamic memory (`std::vector`)
* Custom predicate sorting with `<algorithm>`

## 💻 How to Run
1. Compile the code: `g++ main.cpp -o main`
2. Run the executable: `./main`
